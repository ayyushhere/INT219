Tailwind Css Box Model

Box Model

This box demonstrates the properties of the box model using Tailwind CSS:

Padding

This box has p-4, which means padding of 1rem(16 px)

Border

This box has Border-4, a dashed border with border-blue-300.

Margin

This box is separated from other using mb-4 (Margin-bottom).

Width and Height

This box has w-64 (16rem) and h-32 (8rem)
